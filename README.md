## 간단한 프로젝트 소개  
- 검색 기능 구현  
- 로고 클릭 시 메인 페이지로 이동  
- 비디오 클릭 시 뷰 페이지로 전환  
- iframe을 활용한 영상 재생  
- 상세 정보 및 관련 영상 목록 표시
